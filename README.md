# TaskFlowandStatusTracking
Java web项目框架
①　前端：Thymeleaf（html）  
②　后端：Spring Boot + Maven + MyBatis  
③　数据库：MySQL

任务流转与状态跟踪系统
开发一个能够追踪任务状态的系统。如学习任务管理系统、团队协作任务系统、社团任务系统等。
（一）用户端功能：
① 用户注册、登录、验证码验证、安全退出。
② 新建任务（支持图片上传）。
③ 查看任务状态（如待处理/进行中/已完成）。
④ 对任务进行留言或备注。
⑤ 查看个人任务完成情况统计。
（二）管理员端功能：
① 管理员登录与后台管理功能。
② 修改任务状态并查看全部任务列表。
③ 任务分类管理（CRUD）。
④ 按状态、日期、关键字等条件查询任务。
⑤ 查看任务完成率与统计报表。
⑥ 后台操作日志记录。
